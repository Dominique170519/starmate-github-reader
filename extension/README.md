# 星伴读 Chrome 伴读侧栏（实验版）

这是一个不依赖模型 API 的 Manifest V3 扩展原型。

## 当前能力

- 在 GitHub 仓库页面显示 README / Markdown 真实目录。
- 点击目录定位并高亮原文。
- 在当前页面搜索原文，不生成或编造回答。
- 按仓库保存笔记，并把选中的原文加入笔记。
- 一键把当前仓库链接带入星伴读 Web App。

## 本地安装

1. 打开 Chrome 的“扩展程序”页面。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本目录。

这是实验版，正式上架 Chrome Web Store 前还需要补充图标、隐私说明、版本发布和商店审核材料。
